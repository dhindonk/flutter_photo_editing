# Aplikasi Editor Gambar untuk Pembelajaran Pengolahan Citra
## 📌 Konsep Umum
Aplikasi ini ditujukan untuk mahasiswa yang sedang mempelajari pengolahan citra digital. Tujuannya adalah untuk memberikan pengalaman langsung dalam menerapkan teori pengolahan citra ke gambar nyata melalui antarmuka interaktif dan edukatif.

## 🎯 Tujuan Aplikasi
- Mempermudah mahasiswa memahami konsep dasar pengolahan citra.
- Menyediakan alat bantu eksperimen secara visual dan interaktif.
- Menyediakan informasi teori singkat untuk setiap filter/fitur.

## 🧩 Fitur Berdasarkan Materi Kuliah

### 1. Jenis Citra
- Konversi RGB ke Grayscale
- Konversi RGB ke Citra Biner (manual threshold & Otsu)

### 2. Operasi Pixel dan Histogram
- Brightness & Contrast Adjustment
- Histogram RGB dan Grayscale
- Histogram Equalization

### 3. Operasi Ketetanggaan & Konvolusi
- Konvolusi dengan kernel custom (3x3)
- Filter Rerata (Mean), Median, dan Deteksi Batas Manual

### 4. Operasi Geometri
- Translasi (geser gambar)
- Rotasi (dengan sudut bebas)
- Scaling (perbesar/perkecil)

### 5. Morfologi Citra
- Erosi
- Dilasi
- Opening
- Closing

### 6. Segmentasi Citra
- Deteksi Tepi:
  - Roberts
  - Prewitt
  - Sobel
  - Canny
- Deteksi Garis (misalnya Hough Transform)

### Struktur Modul UI
Setiap modul memiliki struktur serupa:
```
+------------------------------------------+
| [Judul Modul: misal "Filter Median"]     |
+------------------------------------------+
| ⓘ Penjelasan Teori (bisa di-toggle)      |
| --------------------------------------   |
| Parameter: [slider / input box]          |
| [🔘 Terapkan Filter]                     |
+------------------------------------------+
| Before-After View                        |
+------------------------------------------+
```

### Contoh Modul: Filter Median
- Judul: Filter Median
- Info Teori: Penjelasan tentang bagaimana filter ini bekerja (mengambil nilai tengah dari pixel sekitar)
- Parameter: Ukuran kernel (3x3, 5x5, dst)
- Output: Tampilan gambar sebelum dan sesudah

## 🔄 Alur Penggunaan Aplikasi
1. Load gambar dari galeri atau kamera
2. Pilih modul sesuai topik pembelajaran
3. Atur parameter jika diperlukan
4. Terapkan filter dan lihat hasilnya
5. Baca info teori untuk pemahaman
6. Simpan hasil atau ekspor laporan

## 🧠 Nilai Edukasi Tambahan
- Penjelasan teori interaktif per modul
- Eksplorasi kernel custom
- Simulasi efek morfologi dengan visualisasi structuring element
- Laporan otomatis berisi: nama filter, parameter yang digunakan, hasil visual

## 📚 Kesimpulan
Aplikasi ini menyatukan antara teori dan praktik dalam pengolahan citra, menjadikannya sarana pembelajaran yang menyenangkan, efektif, dan mendalam untuk mahasiswa teknik informatika atau bidang terkait.

## 🚀 Implementasi Teknologi (Flutter)
Aplikasi ini dibangun menggunakan Flutter karena mendukung:
- Performansi tinggi untuk manipulasi gambar
- Antarmuka lintas platform (Android/iOS/Web/Desktop)
- Fleksibilitas UI untuk membuat canvas custom

### 📦 Library/Plugin Pendukung
- `image` → manipulasi gambar (filter, histogram, dsb)
- `flutter_hooks` atau `riverpod` → state management modular
- `flutter_custom_paint` → menggambar canvas interaktif
- `photo_view` atau `interactive_viewer` → zoom & pan pada canvas
- `path_provider` + `image_picker` + `file_saver` → akses gambar & penyimpanan

## 🖼️ Desain UI + Canvas Interaktif
### 🎨 Konsep Canvas Seperti Figma
- Canvas memiliki **ukuran tak terbatas** (scrollable dan pannable area)
- Gambar yang di-upload akan ditempatkan sebagai **widget yang bisa diproses**
- Hasil pengolahan citra ditampilkan sebagai **lapisan (layer) baru**
- Mahasiswa bisa menambahkan **catatan kecil (sticky notes)** sebagai anotasi teori
- Bisa menggeser layer ke area kosong canvas, mirip artboard

### 🧭 Contoh Layout UI Modern (Flutter)
```
+--------------------------------------------------------------------------------+
|  AppBar: [🏠 Beranda] [📚 Modul] [⚙️ Pengaturan] [💾 Simpan]                     |
+--------------------------------------------------------------------------------+
| Sidebar:                                                                      |
| [📂 Load Image]                                                               |
| [📁 Jenis Citra]                                                              |
| [🎛 Filter]                                                                   |
| [🔄 Transformasi]                                                             |
| [🧲 Morfologi]                                                                |
| [🎯 Segmentasi]                                                               |
+--------------------------------------------------------------------------------+
|                               Canvas Area (zoomable)                          |
|  ┌────────────────────────────────────────────────────────────────────────┐  |
|  |   Gambar original + layer hasil filter + anotasi + pengaturan kernel  |  |
|  |   Bisa drag, zoom, scroll seperti Figma                               |  |
|  └────────────────────────────────────────────────────────────────────────┘  |
+--------------------------------------------------------------------------------+
| Bottom Panel: [Parameter Kernel] [Undo/Redo] [Export Laporan]                |
+--------------------------------------------------------------------------------+
```

## ✨ Fitur Kreatif Tambahan
### 📌 Anotasi Edukasi
- Mahasiswa bisa klik satu layer, lalu klik ikon "ⓘ" untuk melihat:
  - Teori singkat
  - Formula matematika
  - Gambar visualisasi kernel/mask

### 🔁 History Visual
- Setiap tindakan disimpan sebagai "langkah" yang bisa di-replay atau dikembalikan (Undo/Redo)
- Bisa mengekspor semua langkah menjadi laporan PDF (berisi gambar, filter yang dipakai, parameter)

### 🎞️ Simulasi Proses
- Mahasiswa bisa menjalankan proses **step-by-step** untuk melihat efek konvolusi, morfologi, dll
- Cocok untuk memperagakan algoritma saat presentasi

### 🧪 Eksperimen Mandiri
- Mahasiswa bisa mengganti kernel konvolusi sesuka hati
- Bisa memasukkan rumus deteksi edge manual

## 🧠 Kesimpulan Tambahan
Dengan pendekatan canvas seperti Figma, aplikasi ini tidak hanya menjadi alat praktis untuk pengolahan citra, tetapi juga dapat mendorong kreativitas mahasiswa dalam memahami dan memvisualisasikan konsep teoritis secara langsung dan interaktif.
